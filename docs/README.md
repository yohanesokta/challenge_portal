# Dokumentasi Administrator CodeLab JAI

Selamat datang di pusat dokumentasi untuk administrator CodeLab JAI. Halaman ini berisi panduan dan catatan perubahan teknis proyek.

## Daftar Dokumen

- [Panduan Membuat Soal](/docs/tutorial): Tutorial lengkap mengenai cara pembuatan soal, tipe evaluator, dan penggunaan `assert` yang benar.
- [Changelog Proyek](/docs/changelog): Catatan riwayat perubahan fitur, perbaikan bug, dan pembaruan teknis lainnya.

## Catatan Penting
- Semua pengujian menggunakan bahasa **Python**.
- Pastikan untuk selalu memeriksa `changelog` setelah melakukan pembaruan sistem untuk mengetahui fitur-fitur baru yang tersedia (seperti dukungan input `stdin`).
